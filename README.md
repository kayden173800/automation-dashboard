# Python Automation Dashboard

## Overview
A Flask-based backend automation system that allows users to upload CSV files, processes the data using Pandas, and stores results in a SQLite database for persistent tracking.

## Features
- CSV file upload system
- Automated data processing (Pandas)
- Salary analytics (avg, max, min)
- SQLite database storage
- Dashboard for upload history

## Tech Stack
- Python
- Flask
- Pandas
- SQLite
- HTML/CSS

## What I Learned
- Backend request handling
- File upload systems
- Data processing pipelines
- Database integration
- Full-stack Python workflow
